Quando você for pegar uma imagem na internet para usar nos seus sites precisa se lembra dos direitos autorais.

O problema é que algumas imagens não podem ser usadas para fins comerciais, não podem ser editadas. E aí quando você usa isso num site profissional pode te dar dor de cabeça com enchecão de saco das pessoas, não que elas estejam erradas. Mas a chatice vem! 


Por isso, quando você for no google para pesquisar imagens use a ferramenta aprimora de busca de  'direitos de uso':

	Marcadas para reutilização com modificação → Você pode usar essas imagens comercialmente e até editá-las.
	Marcadas para reutilização → você pode usar no seu site mas não pode alterar essa imagem.
	Marcadas para reutilização não comercial com modificação → Você pode usar e modificar essa imagem que não sejam para fins comerciais.
	Marcadas para reutilização não comercial → Você não pode alterar essa imagem e só pode usar para fins NÃO comerciais.
