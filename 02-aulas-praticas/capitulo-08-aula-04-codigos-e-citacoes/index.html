Código fonte:

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Repare que nós deixamos o trecho acima com uma fonte mono espaçada, isso é, com espaçamentos todos iguais.

Nós podemos fazer isso usando a tag <code> </code>

É importante lembrar que a tag <code> </code> não identa o seu código.

Identando um trecho de código com a tag <pre> </pre>

ANTES DE IDENTAR:

num = int(input("Digite um número: ")) if numero % 2 == 0: printf(f"O número {num} é PAR!") else: printf(f"O número {num} é ÍMPAR!") printf("")

Depois de identar: 

        
        num = int(input("Digite um número: "))
        if numero % 2 == 0:
            printf(f"O número {num} é PAR!")
        else:
            printf(f"O número {num} é ÍMPAR!")
        printf("")

OBS: A tag <pre> </pre> leva em consideração os espaços dados dentro da IDE de desenvolvimento.

TEXTO INVERTIDO:

Há uma tag <bdo> </bdo> que serve para inverter o texto da direta para a esquerda e vice versa.

Há dois parâmetros para essa tag: rtl ou ltr

RTL → Da direita à esquerda

LTR → Da esquerda à direita
