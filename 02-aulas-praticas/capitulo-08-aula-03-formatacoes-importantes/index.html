Principais formatações

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Você pode mudar a cor do seu <mark> usando o style inline ou no cabeçalho.


Texto grande e pequeno

Estamos criando um texto grande e outro texto pequeno nesse parágrafo.


Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.
Para fazer isso, use a tag <dell>.


Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase ao texto, indicando que ali há uma marcação diferente.

Há duas maneiras de fazer isso:
Forma semântica: Use a tag <ins> </ins>
Forma não semântica: Use a tag <u> </u>


Texto sobrescrito

Para inserir coisas do tipo: X20 + 3

Sob é em cima, ou seja, a formatação para para quando colocamos numeros elevados a algum valor, por exemplo.

Nós podemos usar a tag <sup> </sup>. Sup de superior.

X30 + 3X + 5 = 0


Texto subescrito

Para inserir coisas do tipo: H20

Sub é em embaixo, ou seja, a formatação para para quando colocamos fórmulas de química, por exemplo.

Nós podemos usar a tag <sub> </sub>. Sub de inferir, abaixo de.

BCPL30 + PL21 + 5 = 0
