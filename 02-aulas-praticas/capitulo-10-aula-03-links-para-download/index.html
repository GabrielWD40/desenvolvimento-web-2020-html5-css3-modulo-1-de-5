Links para download!

Nessa aula vimos que podemos colocar vários tipos de arquivos para o nosso usuário baixar.

Para ver as regras de sintaxe de cada arquivo, viste a página iana.org e procure pelo tipo de arquivo que você deseja.

Para fazermos o download, indicamos que é um link e depois passamos o parâmetro <download="nome-do-arquivo"> e depois o tipo dele <type="application/tipo de arquivo">
