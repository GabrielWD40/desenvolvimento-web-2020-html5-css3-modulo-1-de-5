Usando Links internos

Clique aqui para abrir a página 002


Para ajudar nos mecanismos de buscas e otimizar o nosso site, podemos usar a propriedade <rel="nex / prev">

A propriedade next siginifica que o conteúdo vai avançar dentro do seu sistema.

A propriedade prev significa que o conteúdo vai voltar atrás no seu sistema.

Não há necessidade de pedir para o navegador abrir outra página quando o link for interno, senão ficaria horrível!


Você também pode acesar a nossa página de notícias!

Veja que quando queremos o conteúdo de uma pasta que se encontra no mesmo diretório, usamos ./nome_da_pasta/nome_do_arquivo.

Já quando precisamos de um arquivo da pasta anterior, usamos ../nome_da_pasta/nome_do_arquivo


Repare que agora eu coloquei o rel="_self", isso serve para dizer que o link deverá abrir na prória página
